About the Experiment

I've implemented a html page with a grid layout using the container, row and col classes. I've used a css file to make these tags distinct in look: Following is the code in CSS:

body { margin:0px; padding:0px; } #north ,#south{ height:100px; width:800px; background-color:red; clear:both; } #west,#center,#east { float:left; } #west, #east{ height:400px; width:100px; background-color:blue; } #center{ height:400px; width:600px; } #container{ width:800px; margin:auto; } .row{ clear:both; width:780px; } .col-1,.col-2,.col-3,.col-4,.col-5,.col-6,.col-7,.col-8, .col-9,.col-10,.col-11,.col-12{ float:left; } .col-1{ width:65px; } .col-2{ width:130px; } .col-3{ width:260px; }
