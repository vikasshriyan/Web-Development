MongoDB - Count Operation

The count method returns the number of developers in my developer collection.

About the Experiment

MongoDB is a cross-platform document-oriented database. Classified as a NoSQL database, MongoDB eschews the traditional table-based relational database structure in favor of JSON-like documents with dynamic schemas (MongoDB calls the format BSON), making the integration of data in certain types of applications easier and faster. Here I have used the "count" method of MongoDB to return the number of documents in a specific collection.

Code Snippet

When the user hits the /countDeveloper URL, the output is the number of documents in a specific collection. I use the count method in MongoDb to fetch the count of developers in my developer collection.


Javascript Code
app.get('/countDevelopers', function (req, res) {
DeveloperModel.count(function (err, data) {
res.json(data);
});
});
