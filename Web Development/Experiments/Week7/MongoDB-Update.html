About the Experiment

MongoDB is a cross-platform document-oriented database. Classified as a NoSQL database, MongoDB eschews the traditional table-based relational database structure in favor of JSON-like documents with dynamic schemas (MongoDB calls the format BSON), making the integration of data in certain types of applications easier and faster. Here I have used the "findOneAndUpdate" method of MongoDB to return the the updated developer based on a specific query and the field to be updated.

Code Snippet

When the user hits the /updateDeveloper URL, the output is the updated developer with new values ina specific field. In this case the specialization field is updated. I use the findOneAndUpdate method in MongoDb to update a document in the collection which is similar to findAndModify method for db.


Javascript Code
app.get('/updateDeveloper', function (req, res) {
var query = {lastName:'Shriyan'};
DeveloperModel.findOneAndUpdate(query, { specialization: 'HTML' }, function (err, data) {
res.json(data);
});
});
