AngularJS - Calculator using Controller.

AngularJS, commonly referred to as Angular, is an open-source web application framework maintained by Google and a community of individual developers and corporations to address many of the challenges encountered in developing single-page applications. Its goal is to simplify both development and testing of such applications by providing a framework for client-side model–view–controller (MVC) architecture, along with components commonly used in rich internet applications.

In this experiment, I've used the controller function to make a calculator which performs the given operations above dynamically on the HTML page.

Code Snippet

Here, I've initially loaded the AngularJS module using .module function in the js file and linking it using ng-app in the HTML file. Using the app namespace and the controller function, linking the ng-controller calculatorController in the HTML page, I then declare function add() with $scope parameter. Two variables are declared for operations on two numbers collected using ng-model firstNumber and secondNumber. I then parse the given inputs into integers and do the addition operation. I use the same logic for other operations as well.

var app = angular.module("cal",[]) // load the angular module.

app.controller("calculatorController", function ($scope) { //Using the controller function with app namspace
$scope.add = function () { // define the add function
var n1 = $scope.firstNumber; // collect the first number from ng-model firstNumber
var n2 = $scope.secondNumber; //collect the second number from ng-model firstSecond
$scope.output = parseInt(n1) + parseInt(n2); //Convert the given i/p into Integers and add
}

$scope.subtract = function () {
var n1 = $scope.firstNumber;
var n2 = $scope.secondNumber;
$scope.output = parseInt(n1) - parseInt(n2);
}

$scope.divide = function () {
var n1 = $scope.firstNumber;
var n2 = $scope.secondNumber;
$scope.output = parseInt(n1) / parseInt(n2);
}

$scope.multiply = function () {
var n1 = $scope.firstNumber;
var n2 = $scope.secondNumber;
$scope.output = parseInt(n1) * parseInt(n2);
}
});
