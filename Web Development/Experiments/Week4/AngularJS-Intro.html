AngularJS Introduction

AngularJS, commonly referred to as Angular, is an open-source web application framework maintained by Google and a community of individual developers and corporations to address many of the challenges encountered in developing single-page applications. Its goal is to simplify both development and testing of such applications by providing a framework for client-side model–view–controller (MVC) architecture, along with components commonly used in rich internet applications.
You can download the AngularJS from here.
In this experiment, I've implemented a controller function to edit the First and Last name dynamically on the HTML page.

Code Snippet

Here, I've initially loaded the AngularJS module using .module function in the js file and linking it using ng-app in the HTML file. Using the app namespace and the controller function, linking the id Controller in the HTML page, I then declare a function with $scope parameter. A student variable is declared with first and last name initialized to null. Using the $scope, I assign the student expression .student

var app = angular.module("demoController", []) // load the angularJS module
app.controller("Controller", // using the controller function and app namespace
function ($scope) { // using the scope as a parameter
var student = {
first: "",
last : "",
};
$scope.student = student; //Assign student expression to $scope.
console.log("Hello World"); //Printing in console for testing purpose.
});
