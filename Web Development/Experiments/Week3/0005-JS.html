About the Experiment:

JQuery with CSS

Here in this experiment CSS properties of DOM Objects is manipulated using jQuery. Using the css class, we can set properties of the CSS to any HTML tags.

Code Snippet:

I'm using the CSS1 and CSS2 class in the 'p' tag to differentiate what CSS properties I want to apply to both of them. On click, I want to change the CSS properties so I use the .css class to set different properties such as the background color, font-color, etc. On refresh, I reload the page using '.reload' function.

$(document).ready(function ()
{
$(".CSS1").click(function () // p tag with CSS1 class.
{
$(this).css({ // Setting different properties using .css function.
"background-color": "blue",
"color":"white",
"font-style": "italic",
"fontFamily": "Impact,Charcoal,sans-serif"
})
});

$(".CSS2").click(function () // p tag with CSS1 class.
{
$(this).css({ // Setting different properties using .css function
"background-color": "red",
"font-style": "bold",
"fontFamily": "Courier New"
})
});

$(".CSS3").click(function () { // h tag with CSS3 class,
$(".CSS3").css({ // Setting different properties using .css function
"background-image": 'url("http://upload.wikimedia.org/wikipedia/en/c/cc/Chelsea_FC.svg")',
"width": "210px",
"height": "210px",
"font-size": "0"
})
});

$(".CSS4").click(function () { //h tag with CSS4 class
$(".CSS4").css({ // Setting different properties using .css function
"background-image": 'url("http://upload.wikimedia.org/wikipedia/en/5/53/Arsenal_FC.svg")',
"width": "322px",
"height": "379px",
"font-size": "0"
})

$('#refresh').click(function () // Using the refresh id of the input button.
{
location.reload(); // Reload the page.
});
});
