About the Experiment

Accordion-Sortable elements

The experiment is performed to implement Accordion-Sortable elements on HTML page using jQuery-UI which can be downloaded from here. The Accordion is used to expand/collapse content which is broken into sections much like tabs. This happens when the header of the tab is clicke, in this case the h3 tag. The Sortable is used to sort the elements by choice, it shares the draggble properties meaning the element can be dragged to a specific location in the order.

JQuery Script:
$(function ()
{
$("#accordion")
.accordion({
header: "> div > h3" // Set the header inside the div tag.
})
.sortable({
axis: "y", // Sort vertically
handle: "h3", // Sort the header tag.
stop: function (event, ui)
{
$(this).accordion("refresh"); // Refresh accordion to handle new order
}
});
});
