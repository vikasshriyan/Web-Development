About the Experiment

Resizing Image

The purpose of this experiment is to write a simple javascript that would increase/decrease the size of an image as per the zoom percentage selected by the user.

Code Snippet

Initially, the size of the image is 300px and it's id is image. We have a HTML range controller with id="zoomController" which is used to get the percentage zoom from the user.
The zoomSelector can shift between 0% to 100% with 1% increase on each step. The onchange event of the zoomController is set to call a javascript function sizechange().

function sizechange() {
var img_height = 300; //Original height of the image in pixels
var step = document.getElementById('zoomSelector').valueAsNumber; //Get the current zoom value from the user by using document.getElementId()
var heightChange = img_height * (step / 100); //Calculating the percentage increase and decrease
img_height = img_height + heightChange; //Setting the new value of the image
document.getElementById('image').height = img_height;
document.getElementById('currentSize').innerText = step + '%'; //Display the current zoom percentage to the user.
}
