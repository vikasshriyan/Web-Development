Description

Demonstrate the idea of hiding and showing objects, in this case an image on button click
Click on the hide button to hide the image and show button to bring back the image.

Demonstration


Click on the hide button to make the image disappear. And then click on the show button to make the image reappear

About the Experiment

Fade and Show

In this experiment, when you click on the hide button, the picture below fades away and on clicking on show button makes back the picture comes back. The experiment is done using JQuery. Jquery is basically used to enable client side dynamic content.

I'm using the fadeToggle function with parameter of 3 seconds to make the image fade away and come back. I'm targeting the img tag in the html code and the buttons with hide and show id's in the following function.
$(document).ready(function ()
{
$("#hide").click(function ()
{
$("img").fadeToggle(3000);
});
$("#show").click(function ()
{
$("img").fadeToggle(3000);
});
});
