About the Experiment

Draggable objects

The experiment is performed to implement draggable items on HTML page using jQuery-UI which can be downloaded from here. The draggable function when called with a particular id allows that object to be dragged around. The draggable function allows the object to be dragged horizontally or vertically by specifying x or y axis respectively.

JQuery Script
$('#drag-me').draggable(); // drag all over the page
$('#drag-me-hor').draggable({ axis: "x" }); // drag horizontally on x-axis
$('#drag-me-ver').draggable({ axis: "y" }); // drag vertically on y-axis
$('#drag-me-grid').draggable({ grid: [25, 25] });
