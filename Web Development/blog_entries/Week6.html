NodeJS and OpenShift

During the sixth week, Prof. Jose started with a new technology from the MEAN Stack called NodeJS. During the two lectures, prof. taught the very basics of NodeJS. Nodejs is an open source, cross-platform runtime environment for server-side and networking applications. Nodejs applications are written in JavaScript, and can be run within the Node.js runtime on OS X, Microsoft Windows, Linux and FreeBSD. Nodejs provides an event-driven architecture and a non-blocking I/O API that optimizes an application's throughput and scalability. These technologies are commonly used for real-time applications.
